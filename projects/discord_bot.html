This project is just a simple discord bot that I made for fun.

Github: here

Commands

I created several commands with this bot. The commands are triggered by prefixing a . (dot).

  1. help help, I need somebody help
  2. split help, I need somebody help
  3. ping help, I need somebody help
  4. 8ball <question> help, I need somebody help
  5. stopspam help, I need somebody help
  6. allowspam help, I need somebody help
  7. spam <time> <content> help, I need somebody help
  8. guessnum help, I need somebody help
  9. guessnum <maximum number> help, I need somebody help
  10. remind <time> <reminder> help, I need somebody help

This is a fun little project, the guessnum command was called the most. I might add more commands in the future.
